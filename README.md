# PKU beamer template lightweight designed

这是北大风格的Beamer模板，在原有模板[PKU-Beamer-Theme](https://github.com/inFaaa/PKU-Beamer-Theme)基础上进行了风格调整和轻量化优化，加入了更多北大配色。当前版本暂时仅支持英文，中文使用请加入ctex包并使用XeLatex编译，但字体需要进一步调整
